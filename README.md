## nakashun-com website

### Development

#### Docker

Avilable on Docker 🐳

``` bash
# Build Dockerfile
$ docker build . -t nakashun-com
```

or

``` bash
# Using docker-compose.yml
$ docker-compose up
```

#### Local

``` bash
# install dependencies
$ yarn install
$ yarn dev
```

### Environment

🐳 Docker
🌏 Nuxt 2.4^
