# Portfolio-namoo-san

Deploy on netlify. (Public)

## Website my portfolio

### Deploy

Using netlify.

### Environment

Nuxt 2.4^

> My portfolio website for netlify.

## Build Setup

``` bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn run dev

# build for production and launch server
$ yarn run build
$ yarn start

# generate static project
$ yarn run generate
```
